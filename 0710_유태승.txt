0709 쪽지 시험 문제 풀이

1. 객체(인스턴스)는 프로그램 내에 실존하는 것이며, 클래스는 이를 실체화하기 위한 도면이다.

2. 자식 클래스는 부모 클래스를 받아들일수가 없다. = 암시적으로 자식 클래스의 형식을 통해 부모 클래스를 개체화하는 것이 불가능 하다.

3. this 키워드는 실체화된 자시 자신을 가리키는 것이며, base 키워드는 자신이 상속받고 있는 부모 클래스를 가리키는 것이다.

4. 구조체는 값 형식이라 얕은 복사가 아닌 깊은 복사가 일어나며, 참조 형식이 아닌 값 형식이다.

5. readonly 안에서는 target 즉 읽기 전용이라 값을 변경 시키는 것은 불가능 하기에,
return curr.. * 1.8 + 32; 으로 curr..의 값을 직접 변경시키지 않고 값을 출력 시켰다.
